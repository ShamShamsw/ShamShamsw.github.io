Trigger a Job Based on Labels

A showcase of GitHub Actions and Azure integration.

Description

This project demonstrates how to use GitHub Actions to trigger workflows based on labels, enabling automated deployments to Azure environments.

Objectives Completed

  • Configured GITHUB_TOKEN permissions for workflows.
  • Set up triggers for GitHub Actions workflows based on pull request labels.
  • Built and pushed Docker images to GitHub Container Registry.
  • Deployed application containers to Azure Web Apps.
  • Managed Azure credentials securely using GitHub Secrets.
