Bias Detection in Income Prediction Model

Exploring bias in machine learning for better decision-making.

Description

This project is part of a class assignment focused on identifying and interpreting bias in machine learning models. Using the Aequitas toolkit, we analyze potential educational bias in an income prediction model trained on 1994 U.S. Census data. The goal is to evaluate how bias may impact business decisions, such as selecting a location for a new boba tea store. Some information is redacted for privacy reasons.

Objectives Completed

  • Imported and preprocessed the "US Adult Income Data" dataset.
  • Set education level as the protected attribute with "Bachelors" as the reference group.
  • Selected fairness metrics: False Positive Rate Parity and False Negative Rate Parity.
  • Generated a Fairness Report using Aequitas with a disparity intolerance of 80%.
  • Interpreted the results to assess bias and its potential impact on business decisions.
