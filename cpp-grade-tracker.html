CPP Grade Tracker

Program manages and tracks student grades.

Description

The CPP Grade Tracker is a console-based application designed to help educators manage student grades efficiently. It allows adding, removing, and displaying grades for individual students.

Objectives Completed

  • Implemented a Student class to manage individual student data.
  • Created a GradeManager class to handle multiple students and their grades.
  • Added functionality to add and remove students and grades dynamically.
  • Provided a method to display all students and their grades.
  • Demonstrated the functionality with a sample main program.
  • Ensured proper error handling for invalid inputs.
  • Optimized the program for scalability with larger datasets.
  • Documented the code for better readability and maintenance.
