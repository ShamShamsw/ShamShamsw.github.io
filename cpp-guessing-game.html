C++ Guessing Game

A fun and interactive number guessing game.

Description

This project is a simple C++ program where the user tries to guess a randomly generated number between 1 and 100. The program provides feedback on whether the guess is too high or too low and keeps track of the number of attempts.

Objectives Completed

  • Implemented random number generation using rand() and srand().
  • Created a loop to handle user input and provide feedback.
  • Tracked the number of attempts made by the user.
  • Added functionality to allow the user to play again.
