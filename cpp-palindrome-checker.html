C++ Palindrome Checker

A simple program to check if a string is a palindrome.

Description

This project is a C++ program that determines whether a given string is a palindrome. It uses standard library functions to reverse the string and compare it with the original input.

Objectives Completed

  • Implemented a function to reverse a string and check for equality.
  • Used standard input and output for user interaction.
  • Demonstrated basic string manipulation and algorithm usage in C++.
