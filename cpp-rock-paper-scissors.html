Rock, Paper, Scissors Game

A fun and interactive game built with C++.

Description

This project is a simple Rock, Paper, Scissors game implemented in C++. It allows the user to play against the computer, which makes random choices. The program determines the winner and displays the result.

Objectives Completed

  • Accept user input for Rock, Paper, or Scissors.
  • Generate a random choice for the computer.
  • Compare user and computer choices to determine the winner.
  • Handle invalid inputs gracefully.
  • Display the results of the game to the user.
