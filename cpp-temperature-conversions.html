C++ Temperature Converter

A simple and efficient tool for temperature conversions.

Description

This project is a C++ program that allows users to convert temperatures between Fahrenheit, Celsius, and Kelvin. It demonstrates the use of functions, conditional statements, and user input handling in C++.

Objectives Completed

  • Implemented a user-friendly interface for temperature input and unit selection.
  • Developed functions to handle temperature conversions between Fahrenheit, Celsius, and Kelvin.
  • Ensured error handling for invalid unit inputs.
  • Showcased modular programming by separating input, processing, and output logic.
