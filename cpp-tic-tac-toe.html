C++ Tic-Tac-Toe

A simple console-based Tic-Tac-Toe game.

Description

This project is a console-based implementation of the classic Tic-Tac-Toe game written in C++. It demonstrates the use of 2D vectors, loops, and conditional logic to create an interactive game for two players.

Objectives Completed

  • Implemented a function to display the Tic-Tac-Toe board.
  • Created logic to check for a winner in rows, columns, and diagonals.
  • Handled player input and validated moves.
  • Implemented a turn-based system for two players.
  • Displayed the game result (win or draw) at the end.
