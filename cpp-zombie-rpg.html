C++ Zombie RPG

Survive the apocalypse with strategy and skill.

Description

This project is a text-based zombie survival RPG written in C++. Players navigate through various locations, manage resources, and fight zombies to survive. The game demonstrates object-oriented programming principles, random number generation, and interactive gameplay mechanics.

Objectives Completed

  • Implemented player stats and inventory management.
  • Created a map with multiple locations and random events.
  • Designed a combat system for fighting zombies.
  • Utilized object-oriented programming for modularity and clarity.
  • Incorporated random number generation for dynamic gameplay.
