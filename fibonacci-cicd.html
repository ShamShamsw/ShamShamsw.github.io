Fibonacci CI/CD

Automating Fibonacci sequence generation with CI/CD pipelines.

Description

This project demonstrates the implementation of a Fibonacci sequence generator in Python, complete with unit tests and a CI/CD pipeline for automated testing and deployment.

Objectives Completed

  • Developed a Python function to calculate Fibonacci numbers.
  • Created unit tests to validate the functionality of the Fibonacci generator.
  • Set up a CI/CD pipeline to automate testing and deployment.
  • Hosted the project on GitHub for version control and collaboration.
