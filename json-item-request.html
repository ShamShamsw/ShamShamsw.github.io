Item Request Program

A simple program to request items and manage user information.

Description

This program demonstrates the use of getter and setter methods, private and public arrays, and user interaction in C#. Users can enter their personal information, select an item from a predefined list, and view the details of their request.

Objectives Completed

  • Implemented private fields for user information (First Name, Last Name, Room Number, Hawk Card Number).
  • Created a public array for item selection.
  • Developed methods to gather user input and display requested information.
  • Provided functionality to view requested items and notify users when items are available.
