Machine Learning Analysis in Retail

Class work showcasing machine learning fundamentals in retail analysis.

Description

This project demonstrates the application of basic machine learning methods for retail analysis using global food prices data. Note: Some details are classified as this was class work, and the repository is private.

Main Objectives

  • Analyze retail data to identify trends and patterns.
  • Apply machine learning techniques to predict future outcomes.
  • Evaluate the performance of different machine learning models.

Approaches Used

  • Data preprocessing and cleaning.
  • Exploratory data analysis (EDA).
  • Implementation of regression and classification models.
  • Model evaluation using metrics like accuracy and RMSE.
