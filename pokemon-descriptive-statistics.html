Pokemon Descriptive Statistics

Analyzing Pokemon stats with Python and data visualization.

Description

This project involves analyzing a dataset of Pokemon statistics using Python. It includes data cleaning, visualization, and statistical analysis to uncover insights about Pokemon attributes such as attack, defense, and speed.

Objectives Completed

  • Loaded and cleaned the Pokemon dataset, handling missing values.
  • Performed descriptive statistics and visualized data distributions.
  • Created scatter plots and linear regression models for individual stats.
  • Generated a correlation matrix and heatmap for statistical relationships.
  • Ranked Pokemon by stats and identified top performers in each category.
  • Compared Legendary vs Non-Legendary Pokemon attributes.
