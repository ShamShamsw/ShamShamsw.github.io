Python Baseball Analytics

Analyzing Baseball Data Using Python.

Description

This project involves analyzing baseball data using Python to compare team performance at home and away games. Statistical tests and visualizations are used to uncover insights about scoring patterns.

Objectives Completed

  • Loaded and filtered baseball data for selected teams (MIL, BOS, NYY, LAD).
  • Separated data into home and away games for analysis.
  • Performed t-tests to compare scoring patterns at home and away ballparks.
  • Created visualizations such as boxplots to illustrate scoring distributions.
