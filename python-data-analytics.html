Python Data Analytics

Analyzing sales data with Python for actionable insights.

Description

This project demonstrates the use of Python to analyze quarterly sales data. It calculates total annual sales, average quarterly sales, and identifies the best and worst performing quarters.

Objectives Completed

  • Calculated total annual sales revenue: $120,000
  • Calculated average quarterly sales revenue: $30,000
  • Identified the best quarter: Q4 ($35,000)
  • Identified the worst quarter: Q1 ($25,000)
