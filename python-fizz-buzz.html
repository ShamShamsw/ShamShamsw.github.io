FizzBuzz Program

A comparative implementation in C++ and Python.

Description

This project demonstrates the implementation of the classic FizzBuzz problem in both C++ and Python. It showcases the differences in syntax and approach between the two programming languages while achieving the same functionality.

Objectives Completed

  • Implemented FizzBuzz logic in C++ using arrays and loops.
  • Implemented FizzBuzz logic in Python using lists and loops.
  • Generated random numbers to test the FizzBuzz logic in both implementations.
  • Displayed results for numbers divisible by 3 (Fizz), 5 (Buzz), and both (FizzBuzz).
