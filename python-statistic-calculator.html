Python Statistics Calculator

A comprehensive tool for statistical analysis.

Description

This project is a Python-based statistics calculator that provides a wide range of statistical computations, including mean, median, mode, standard deviation, variance, and more. It is designed to help users analyze data efficiently and effectively.

Objectives Completed

  • Implementation of core statistical functions such as mean, median, and mode.
  • Advanced computations including standard deviation, variance, and z-scores.
  • Support for data visualization concepts like box plots and five-number summaries.
  • Integration of statistical tests such as correlation, regression, and hypothesis testing.
