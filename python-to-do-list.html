To-Do List

Organize your tasks efficiently and effectively.

Description

This project is a dynamic To-Do List application that allows users to manage their tasks with ease. It includes features such as adding new tasks, marking tasks as completed, and editing task details.

Objectives Completed

  • Implemented a table to display tasks with columns for task name, status, and due date.
  • Added functionality to create new tasks dynamically.
  • Enabled editing of task names and due dates directly in the table.
  • Included a "Completed Tasks" section to track finished tasks.
  • Provided options to mark tasks as done, working on it, or not done.
