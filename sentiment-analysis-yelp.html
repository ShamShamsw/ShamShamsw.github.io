Sentiment Analysis of Yelp Business Reviews

Using NLP and machine learning to analyze customer feedback.

Description

This project involves designing a method to analyze Yelp reviews, providing insights into businesses and categorizing reviews as positive or negative. It leverages Natural Language Processing (NLP) and machine learning techniques.

Objectives Completed:

  • Understood the theory behind sentiment analysis, including feature extraction and tokenization.
  • Explored the Yelp Business Reviews dataset for text cleaning and vectorization.
  • Created Word Count and Word Cloud plots to extract meaningful insights.
  • Built a Multinomial Naive Bayes Classifier to classify reviews as positive or negative.
