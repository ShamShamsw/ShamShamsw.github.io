Odysseus Boat and Gear Rental

SQL practice to create a data management type project.

Description

This project demonstrates the creation of a database system for managing a boat and gear rental business. It includes tables for products, warehouses, shipments, orders, and vendors, showcasing relationships through foreign keys and efficient data handling.

Objectives Completed

  • Designed and implemented a relational database schema.
  • Created tables for products, warehouses, shipments, orders, and vendors.
  • Established relationships using foreign keys.
  • Inserted sample data for testing and validation.
  • Executed queries to retrieve and manipulate data effectively.
